# CppToolKit

## CppToolKits目标
集合C++底层各种常用的数据结构，常用工具，日志，常用算法，多线程，常用设计模式和网络模块。为开发提供稳定可靠的工具库。

## CppTollKits特点
- 基于C++11开发，，无任何其他依赖
- 代码尽量清晰简洁，稳定可靠
- 提供多样的工具，尽量满足日常需求使用
